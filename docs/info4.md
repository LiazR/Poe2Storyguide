1. #### 金司馬區(君锋镇)

岛屿探索无固定顺序.。而且0.5版本更新会改动流程

  对话沿途感叹号Npc，右下角对话马克鲁前往凱吉灣

2. #### 凱吉灣(落锚湾)

赛季玩法奖励：崇高石   

 找到亡者之箱，获得撕裂的地图碎片

  (可选)**低阶工匠石**。一艘废弃船开宝箱掉落(倒计时结束会被秒杀)

  找到并进入旅程之末

3. #### 旅程之末(旅途终点)

赛季玩法奖励：点金石  

找到击杀boss哈特林，获得维金，回金司馬區，对话丹宁，获得维金长钉。
  返回旅程之末传送点，对话npc哈特林，击杀恐惧具象，打完后回城找图金获取**2点剧情天赋**

  ```json
  "badge": { "text": "+2天赋", "type": "`skill`" },
  ```

  返回船上，对话马克鲁前往伯金氏島

4. #### 金氏島(亲眷岛)

赛季玩法奖励：宝石匠棱镜  

矿场箱子可以开出低阶工匠石.

  地图边缘找到水手的遗体，点击获取撕裂的地图碎片

  此处boss击杀有高级符文+11级宝石+4级辅助宝石

  找到火山迷窟进入

5. #### 火山迷窟(熔火院落)

赛季玩法奖励：4级辅助宝石  

贴边跑到BOOS房间，干掉BOOS金氏領主·克魯托克

  返回船上，对话马克鲁前往瓦卡帕努島

6. #### 瓦卡帕努島(瓦卡帕努岛)

赛季玩法奖励：工匠石  

找到石化海盗，点击获取撕裂的地图碎片

  鲨坑boss击杀获得鲨鱼皮，对话凯玛纳获取技能宝石，鲨鱼任务可做可不做

  找到并进入吟謠洞窟

7. #### 吟謠洞窟(冥曲洞窟)

赛季玩法奖励：随机护符  

点击唤声蛤蜊获得嗡鸣珍珠，第四章完成后金司馬區npc对话获取珍珠项链，可做可不做
  击杀boss戴娅摩拉，对话兜帽密师
  返回船上，对话马克鲁前往廢棄監獄

8. #### 廢棄監獄(废弃监牢)

赛季玩法奖励：崇高石 

 刷怪掉落教堂钥匙，找到并打开教堂大门，激活正义女神选择任意药水BUFF，可随时切换属性

  ```json
  "badge": { "text": "药水buff", "type": "`loot`" },
  ```

  地图边缘找到單獨禁閉

9. #### 單獨禁閉(遗世监禁)

赛季玩法奖励：随机符文  

尽头找到boss囚徒，boss倒地的时候启动旁边开关才可以继续造成伤害

  返回船上，对话马克鲁前往伯勞鳥之島

10. #### 伯勞鳥之島(百舌岛)

	赛季玩法奖励：4级辅助宝石

  找到尸骸巢穴，获得撕裂的地图碎片

  找到腐化之巢穴，击杀boss切扎(0.5版本更新会改动)0.4更新后记得救NPC才能解锁去悉尼寇拉之眼！

  返回船上，对话马克鲁前往阿拉塔斯

11. #### 阿拉塔斯(阿拉斯塔)

	赛季玩法奖励：12级技能宝石

	走出城镇后悬崖边缘处有3崇高记得领取(城镇中有3富豪实在找不到就算了)

	击杀boss拖维安，找到并进入挖掘

12. #### 挖掘(挖掘场)

赛季玩法奖励：随机稀有护身符   

击杀boss首席使者，往里面走等动画结束

   返回船上，对话罗德里前往尼加卡努

13. #### 尼加卡努(纳卡努)

  赛季玩法奖励：高阶工匠石

   找到并进入部族之心

14. #### 部族之心(部族之心)

赛季玩法奖励：12级精魂宝石   

击杀BOSS塔瓦凯，必须等动画结束，回金司馬區，对话兜帽密师第四章完成

   

15. #### 悉妮蔻拉之眼(辛格拉之眼)

赛季玩法奖励：混沌石

   ###### 获取最大魔力属性

   完成3个试炼，找到沉默之厅，点击感叹号获取5%最大魔力

   ```json
   "badge": { "text": "+5%最大魔力", "type": "`loot`" },
   ```

   完成任务后，往反方向跑，找到亡者之殿进入

16. #### 亡者之殿(亡者之境)

赛季玩法奖励：随机符文  

 完成3个试炼，获取空白文身，点击图腾选择奖励5%抗性或者5属性点都行

   ```json
   "badge": { "text": "+5%抗性*3/+5属性*3", "type": "`loot`" },
   ```

17. #### 祖靈的試煉(祖先的试炼)

   完成最后一个试炼，击杀boss白之雅玛，获取银币，注意往boss房里面走，进入命运试炼对话npc，获取2天赋点书

   ```json
   "badge": { "text": "+2天赋", "type": "`skill`" },
   ```
