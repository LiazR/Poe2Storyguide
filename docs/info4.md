1. #### 金司馬區

	岛屿探索无固定顺序

	对话沿途感叹号Npc，右下角对话马克鲁前往金氏島

2. #### 金氏島

	矿场箱子可以开出低阶工匠石.

	地图边缘找到水手的遗体，点击获取撕裂的地图碎片

	找到击杀boss眼盲亘獸获得2天赋点书

	```json
	"badge": { "text": "+2天赋", "type": "`skill`" },
	```

	找到火山迷窟进入

3. #### 火山迷窟

	贴边跑到BOOS房间，干掉BOOS金氏領主·克魯托克

	对话马提吉，解锁地图悉妮蔻拉之眼，返回船上，对话马克鲁前往悉妮蔻拉之眼

4. #### 悉妮蔻拉之眼

	###### 获取最大魔力属性

	完成3个试炼，找到沉默之厅，点击感叹号获取5%最大魔力

	```json
	"badge": { "text": "+5%最大魔力", "type": "`loot`" },
	```

	完成任务后，往反方向跑，找到亡者之殿进入

5. #### 亡者之殿

	完成3个试炼，获取空白文身，点击图腾选择奖励5%抗性或者5属性点都行

	```json
	"badge": { "text": "+5%抗性*3/+5属性*3", "type": "`loot`" },
	```

6. #### 祖靈的試煉

	完成最后一个试炼，击杀boss白之雅玛，获取银币，注意往boss房里面走，进入命运试炼对话npc，获取2天赋点书

	```json
	"badge": { "text": "+2天赋", "type": "`skill`" },
	```

	返回船上，对话马克鲁前往廢棄監獄

7. #### 廢棄監獄

	刷怪掉落教堂钥匙，找到并打开教堂大门，激活正义女神选择任意药水BUFF，可随时切换属性

	```json
	"badge": { "text": "药水buff", "type": "`loot`" },
	```

	地图边缘找到單獨禁閉

8. #### 單獨禁閉

	尽头找到boss囚徒，boss倒地的时候启动旁边开关才可以继续造成伤害

	返回船上，对话马克鲁前往凱吉灣

9. #### 凱吉灣

	找到亡者之箱，获得撕裂的地图碎片

	找到并进入旅途终点

10. #### 旅程之末

	找到击杀boss哈特林，获得维金，回金司馬區，对话丹宁，获得维金长钉。
	返回旅程之末传送点，对话npc哈特林，击杀恐惧具象
	返回船上，对话马克鲁前往伯勞鳥之島

11. #### 伯勞鳥之島

	找到尸骸巢穴，获得撕裂的地图碎片

	找到腐化之巢穴，击杀boss切扎

	返回船上，对话马克鲁前往瓦卡帕努島

12. #### 瓦卡帕努島

	找到石化海盗，点击获取撕裂的地图碎片

	鲨坑boss击杀获得鲨鱼皮，对话凯玛纳获取技能宝石，鲨鱼任务可做可不做

	找到并进入吟謠洞窟

13. #### 吟謠洞窟

	点击唤声蛤蜊获得嗡鸣珍珠，第四章完成后金司馬區npc对话获取珍珠项链，可做可不做
	击杀boss戴娅摩拉，对话兜帽密师
	返回船上，对话马克鲁前往阿拉塔斯

14. #### 阿拉塔斯

	击杀boss拖维安，找到并进入挖掘

15. #### 挖掘

	击杀boss首席使者，往里面走等动画结束

	返回船上，对话罗德里前往尼加卡努

16. #### 尼加卡努

	找到并进入部族之心

17. #### 部族之心

	击杀BOSS塔瓦凯，必须等动画结束，回金司馬區，对话兜帽密师第四章完成
