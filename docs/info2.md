1. #### 瓦斯提里郊區(瓦斯提里外沿)

  跑图贴边走，找BOOS撕裂者

  打完BOOS回城交任务，拿技能石奖励，进入车队

2. #### 阿杜拉車隊(阿杜拉的商队)

  对话丝克玛艾萨拉，使用沙漠地图移动车队到哈拉妮關口对话

3. #### 哈拉妮關口(哈拉尼之门)

  这张地图下车，快看到NPC先开传送门，往前走NPC有感叹号就直接回城不用跟他墨迹说话

4. #### 莫頓挖石場(莫丹采石场)

  这张图贴边跑多看地图的记录点，一直跑到莫頓礦坑

5. #### 莫頓礦坑(莫丹矿场)

  击败恐懼工程師·魯買，解救笼子里的人，必须对话，
  回阿杜拉的商队回城交任务传送叛徒之路

6. #### 叛徒之路(叛徒小径)

	这张图有两种布局

	第一种：进入一条弯曲的长路后，以监禁之令石碑作为分界有两条路(一边为boss另一边为下张图)

	第二种：是打完boss后继续沿路走较远处到下张图

	个人建议以监禁之令石碑旁的石头上有符咒帖的为升华试炼boss。之后再沿着boss周围区域跑，如遇死路即直接esc重生走监禁之令石碑另一方向即可

  找到一个被封印的门，击杀boss解锁丝克玛试炼

  和叛徒贝尔巴拉相反方向可以找到哈拉妮關口

7. #### 哈拉妮關口(哈拉尼之门)

  对话艾萨拉，艾萨拉推磨盘的地方一定要把怪清完，注意看艾萨拉有没有一直跟随。
  没有的话回到艾萨拉推磨盘的地方清怪即可，之后击杀boss。
  下楼右走激活记录点回阿杜拉車隊

8. #### 絲克瑪試煉(丝克玛试炼)

  升华1：自己打就行

小房间尽量别选铁人试炼(机关迷宫图)

boss是中远距离输出就好了，注意别挨着地上岩浆(掉荣耀值，掉完就被秒杀)

最后宝箱尽量开武器或神秘宝箱，开出好的稀有武器基本一路碾压

  ```json
  "badge": { "text": "升华1", "type": "`loot`" },
  ```

  升华试炼结束之后，根据自己的流派选择升华职业

9. #### 凯斯城(克斯)

  任务道具：卡巴拉部落的圣物

  凯斯城这个系列任务里面，通过刷小怪获得之后的任务道具是卡巴拉部落的圣物

  ##### 专精之书（技能点）-打败BOOS異界·干擾女王·卡巴拉

  卡巴拉也是有点技巧的，比如说你看这个地图有一个怪圈，多注意一下它，它里面就是这个boss。

  ```json
  "badge": { "text": "+2天赋", "type": "`skill`" },
  ```

10. #### 失落之城(失落之城) 

	这张地图比较大且随机，还乱，只能贴边走是最好的选择。如果感觉走错了，直接卡记录点，重新选一个方向再走。

	甲壳虫精英必掉珠宝可做可不做

11. #### 掩埋神殿(掩埋的殿堂)

	路过守卫森严的石箱获得辅助技能宝石，可做可不做

	元素神殿，选择对应属性的祭品获得对应属性的抗性戒指，建议选闪电抗性戒指(第二章最后boss能用)

	进入凱斯之心

	击败这个遗忘之子·阿隆里恩，点燃雕像之后，这个任务完成。回城

12. #### 乳齒象惡地(长毛象荒原 )

	这张地图我们要先找到骨坑，然后再原路返回，再找到无光通道。不解锁深渊工作台的话直接前往骨坑

	地图边缘找到骸骨神殿，点击获得1级辅助技能宝石，可做可不做

13. #### 無光通道(暗光走廊)

	这张图贴边走，找到靈魂深幷。

14. #### 靈魂深幷(灵魂之井)

	解锁深渊工作台

	```json
	"badge": { "text": "解锁深渊工作台", "type": "`loot`" },
	```

15. #### 骨坑(遗迹深坑)

	任务道具：太阳部落的圣物

	通过刷小怪获得之后的任务道具是太阳部落的圣物

	之后找最终boss黑古坑，击杀后回城

16. #### 泰坦之谷(巨人之谷)

	地图边缘激活3个古代封印（有时候不在边缘），激活泰坦之谷传送点，
	传送阵后面打开背包，鼠标拖动物品放置到奖章内，点击激活属性，再点击可以更改属性

	拿完buff之后，我们通过记录点传送到一般是地图中央，它会出现一座山，然后我们开启这个山的时候会有一个洞穴，就是我们下一个任务节点，泰坦石窟。

17. #### 泰坦石窟(巨人石窟) 

	沿地图边缘找boss，击杀boss，获得任务物品，回阿杜拉車隊

	然后通过地图传送到叛徒之路之后，到 船头吹响号角

18. #### 戴斯哈(德莎尔) 

	任务道具：战死部下的遗属（+2天赋）

	```json
	"badge": { "text": "+2天赋", "type": "`skill`" },
	```

	回城之后交给夏布林，获得+2天赋

	继续往下跑 找到 悼念之路

19. #### 悼念之路(怀念之阶)

	异变的花瓶点击获取辅助技能宝石，可做可不做。

	这张图没有技巧，贴着边儿一直跑就行，我们找到这个戴斯哈尖塔

20. #### 戴斯哈尖塔(德莎尔的高塔) 

	这张地图没有技巧，沿着边儿直接跑，找到boss干掉他。

	任务奖励：途中找到卡洛翰的姐妹，点击获取10%闪电抗性

	```json
	"badge": { "text": "+10%闪电抗性", "type": "`loot`" },
	```

	回城交任务，调整装备，交任务物品。

21. #### 無畏隊(无畏战车)

	这张地图沿着边儿直接跑，跑到boss终点击败BOOS

22. #### 無畏隊先鋒(无畏战车先锋)

	在交任务的时候，与黑幽魂对完话，可以直接选择回到角色选择界面，卡时间，进行规避这段跑路的过程。
