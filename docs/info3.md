1. #### 風沙沼澤(飞沙沼泽) 

  ##### 工匠石

  工匠石找到这个营地-----欧洛克營火-----击败BOOS，打开箱子

(可选)树根boss掉落9级技能宝石

  然后进入高地神塔營地

2. #### 高地神塔營地(金字塔营地)

  回城后记得在商店更新装备，如果有需要记得替换。

3. #### 叢林遺跡(丛林废墟) 

  这张地图先碰到哪个，我们都可以先打哪个，没有先后顺序。

  它有三个点

  第1点，是我们先碰到了大猩猩，先把大猩猩打死，获得两点技能天赋点。

  第2点，是我们先碰到了剧毒墓穴这个节点，我们可以进去先把永久buff获得下

  第3点，是我们先碰到，庫阿姆的屍體，送稀有腰带（可选）。

  ```json
  "badge": { "text": "+2天赋", "type": "`skill`" },
  ```

  

4. #### 劇毒墓穴(蛇毒地窟)

  （可选）劇毒墓穴里有个任务道具，拿到后回城选择永久增益

  ```json
  "badge": { "text": "永久增益", "type": "`loot`" },
  ```

  

5. #### 感染荒地(滋孽荒原)

  这个地图的作用在于我们跑图踩点

  跑图过程中，我们如果看到这个龍蜥濕地，我们要进去把初始点要踩一下，再回来。

  继续跑图，一直我们找到阿札克泥沼

  ##### 跑图的技巧，我们贴着边走就可以。

6. #### 阿札克泥沼(阿扎卡泥沼)

  ##### 沼泽女巫（+30精魂）

  干掉BOOS沼泽女巫，获得惊魂

初始门和中间大场地和boss点位一般呈现对角线。确认好大场地(记得点亮中间篝火获取buff)后基本能确认处boss点位  

这个boss房刷新的随机性高，所以跑图的时候我们要多注意一下

  ```json
  "badge": { "text": "+30精魂", "type": "`skill`" },
  ```

7. #### 龍蜥濕地(幻缈湿地) 

  这张地图没有技巧，跑到最后boss这里，打败他。获得混沌试炼门票，去混沌试炼踩点回来去

  然后前往吉卡尼的機械迷城

  ```json
  "badge": { "text": "升华2", "type": "`loot`" },
  ```

8. #### 混沌神殿(混沌大殿)

  升华2，看自己情况打不打

9. #### 吉卡尼的機械迷城(佳华尼的机械迷城) 

  地图左右两边找到1个小灵核，放入石碑开门，再找1个小灵核，
  在地图边缘找到boss黑鳄房间，放入石碑开门击杀boss获取10%火焰抗性，
  再找1个小灵核放入石碑，进入吉卡尼的聖域

  ```json
  "badge": { "text": "+10%火焰抗性", "type": "`loot`" },
  ```

10. #### 吉卡尼的聖域(佳华尼的密殿) 

 激活传送点和记录点，找到两台发动机的记录点激活，探明记录点附近黑暗区域，
按ESC在记录点重新刷新中灵核的位置直到地图上显示为止，遇到大型魂灵核心不显示的情况跑远点再跑回来就会
拾取2个中灵核插入2个发电机，回到传送点门口和阿尔瓦对话，点击大型灵魂核心击杀boss，传送到感染荒地

然后用任务物品，大型核心，插入到感染荒地的石阵祭坛中间进入瑪特蘭水道

11. #### 瑪特蘭水道(马特兰水道) 

	这张图没有什么技巧，直接杀怪，跑图，开启机关，然后最后拉闸完成。

	然后回城交任务，进入淹没之城

12. #### 淹没之城(淹没之城)

	这张图跑图没有技巧，

	但是在这张地图，我们要先把两个记录点先踩好，这两个节点不分先后

	举例

	如果，第1个节点是前往污秽顶峰的节点，第2个节点是熔岩宝库的节点。

	那我们就先打熔岩宝库，反之污秽顶峰。

13. #### 熔岩寶庫(熔火宝库)

	##### 重铸台开启任务（击败BOOS）

	```json
	"badge": { "text": "解锁重铸台", "type": "`loot`" },
	```

	

14. #### 污垢頂峰(污秽巅峰)

	这张地图我们只要把boss打败就可以了，期间刷图碰到的蘑菇任务不用做。

	击败boss后传送到淹没之城，开启门，进入科佩克神廟

15. #### 科佩克神殿(科佩克之殿)

	这张地图，因为随机性高，随便跑吧，他的门一般在地图的三个角，我们找到门之后下三层，找到boss房，干掉他

	打完boss后跟npc对话，这期间我们不用等他，直接传送回城，我们可以在城内进入传送门回到过去。

	

16. #### 奥札爾(乌扎尔)

	地图边缘石板路一直走，击杀boss蛇女，往后面走进入阿戈拉，
	这个图和阿戈拉都可以爆奉献之核，建议在阿戈拉刷奉献之核

17. #### 阿戈拉(阿戈拉特)

	任务:找到献祭匕首的祭台，刺穿牺牲之血，找到2点天赋，

	然后跑图找到漆黑密室。

	```json
	"badge": { "text": "+2天赋", "type": "`skill`" },
	```
