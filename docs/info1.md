1. #### 河岸(污流河畔) 

	初始点河岸，迅速拿装备技能石跑到BOOS房，击杀后到达营地

2. #### 皆伐營地(克里费尔营地) 

	进入营地，只接铁匠倫利的任务拿技能石跑到皆伐

3. #### 皆伐(克里费尔) 

  赛季玩法奖励：蜕变石

  ##### 任务物品：凛冬狼的頭顱 冷冻抗性+10

  地图中间击杀boss贝拉，获得加10%冰冷抗性。

  ##### 技能宝石：

  地图边缘神秘营地开启废弃储物箱，获得1级技能宝石。

  ```
  "badge": { "text": "10%冰冷抗性", "type": "skill" },
  ```

  

4. #### 葛瑞爾林(韧木森林) 

  赛季玩法奖励：蜕变石

  ##### 药剂+辅助宝石

  找到小屋获取生命魔力药剂，干掉忘卻女巫·阿瑞涅获取1级辅助宝石

  ##### 踩点+开启任务（尋求烏娜的幫助）

  拿完药剂根据地图形状（因为随机）决定往哪里走，注意箭头标记的左侧记录点要踩开，开启任务后传送赤谷

5. #### 赤谷(红谷) 

赛季玩法奖励：2级技能宝石

  收集三个鐵鑼方尖碑（任务物品，）击败BOOS鐵鋪之王，拿到符文顱帽，传送门回城交任务

  找到铁匠倫利（图3）获取开启葛瑞爾林的任务物品魔符尖刺

  之后回到葛瑞爾林传送点，把魔符尖刺插到树干上的三个符文之印，完成任务回皆伐營地

6. #### 纏縛陰林(阴森网道) 

  赛季玩法奖励：3级技能宝石

  必须对话乌娜，传送到纏縛陰林，必须对话乌娜完成任务，沿着地图边缘走，

  蘑菇人（掉落1级辅助技能宝石）可打可不打，进入不朽帝国之墓

7. #### 不朽帝国之墓(永恒园林)

	赛季玩法奖励：富豪石

	刚进地图不用管这个老头NPC，迅速找到两个墓穴打败BOOS

	墓穴分布据说呈三角形分布，但是不靠谱，自己多找找吧

	**配偶的墓室**中有个守着保险箱的精英掉落1级辅助技能宝石，赛季玩法给项链

	**政務官陵墓**中有个忘却财富掉落1级辅助技能宝石，赛季玩法给低级符文

	

	击杀两个墓穴boss获取任务道具后回到不朽帝国之墓传送点，点击纪念之门打开大门进去击杀boss，进入獵場

8. #### 獵場(猎场) 

	赛季玩法奖励：崇高石

  **1级辅助宝石+低阶符文**。击杀稀有怪树妖必掉落。

  ##### BOOS+专精书（技能书）

  獵埸这张图直接跑步贴边跑踩点后直接找BOOS鴉鈴

  ```
  "badge": { "text": "+2天赋", "type": "skill" },
  ```

  獵場找到奧格姆農地进入并激活传送阵，獵場找到弗雷索恩进入并激活传送阵

9. #### 弗雷索恩(茂棘深林) 

  赛季玩法奖励：1级辅助宝石

  ##### 寶石花顱骨（+30精魂）

  开启3个仪式祭坛并完成，最后再开启大仪式祭坛击杀boss迷雾之王获得+30精魂和4级精魂宝石

  ```
  "badge": { "text": "+30精魂", "type": "skill" },
  ```

  

10. #### 奧格姆農地(欧甘农地) 

	赛季玩法奖励：4级技能宝石

	(顺路遇到可打)麦田圈空地处打完可得4级宝石

	##### 任务道具 烏娜的魯特琴盒

	找到乌娜的家，获得任务物品乌娜的鲁特琴，找到奥格姆村庄进入并激活传送阵，
	回营地和乌娜对话获得2点天赋点书

	```
	"badge": { "text": "+2天赋", "type": "skill" },
	```

	

11. #### 奥格姆村(欧甘村庄) 

	赛季玩法奖励：工匠石

	##### 任务道具 铁匠的空白符文

	找到铁匠的工具（解锁分解台）

	```
	"badge": { "text": "解锁分解台", "type": "loot" },
	```

	之后继续前进击杀boss后解救囚犯，进入宅第壁壘

12. #### 宅第壁壘(庄园城壁) 

	赛季玩法奖励：5级技能宝石

	##### 技能宝石-行刑架

	建议贴边跑路，找到行刑架有技能宝石，，到达奥格姆宅第

13. #### 奥格姆宅第(欧甘庄园)

	赛季玩法奖励：点金石

	击杀boss（+20最大生命值）

	第一层找boss，在靠近第二层的一个房间，击杀后获得最大生命值
	
	```
	"badge": { "text": "+20最大生命值", "type": "skill" },
	```
	
	之后进入第二层，第三层击杀狼王，完成第一章
