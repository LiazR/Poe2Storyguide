# 更新文档 1.1

## 版本概览

本次更新主要优化章节页浏览体验，并补充第四章攻略内容。重点包括节点列表显示修复、第四章流程重排、以及国际服 / 国服节点名称切换功能。

## 新增功能

### 节点名称切换

章节页地图工具栏新增“国际服名 / 国服名”切换按钮，默认显示国际服名称。

切换后会同步影响：

- 左侧“主线流程”节点列表名称
- 中间地图节点下方名称
- 右侧节点详情标题

名称映射参考 `docs/info1.md` 到 `docs/info4.md` 中的节点标题格式，例如：

```text
河岸(污流河畔)
```

其中：

- `河岸` 为国际服名称
- `污流河畔` 为国服名称

## 内容更新

### 第四章攻略重排

根据 `docs/info4.md` 更新 `content/chapters/act4.json`：

- 按 info4 的攻略顺序重排第四章节点
- 保留同名节点原有地图坐标
- 将攻略内容拆分为更清晰的 `description` 与 `steps`
- 根据 info4 中明确标注的奖励补充 `badge`
- 节点名称括号内国服名称不直接写入章节 JSON，而是由名称切换映射统一管理

第四章当前流程为：

```text
金司馬區 → 凱吉灣 → 旅程之末 → 金氏島 → 火山迷窟 → 瓦卡帕努島 → 吟謠洞窟 → 廢棄監獄 → 單獨禁閉 → 伯勞鳥之島 → 阿拉塔斯 → 挖掘 → 尼加卡努 → 部族之心 → 悉妮蔻拉之眼 → 亡者之殿 → 祖靈的試煉
```

## 体验优化

### 左侧列表滚动修复

修复章节页左侧栏在节点较多时最后一个节点显示不全的问题。

调整后：

- 左侧栏使用固定高度外壳
- 主线流程区域独立滚动
- 节点数量较多时不会被外层裁切
- 最后一个节点可以完整滚动显示

### 章节页布局稳定性

优化左侧栏内部 flex 布局，避免不同章节节点数量差异导致列表高度计算异常。

## 技术调整

新增 `src/data/nodeNames.ts`，集中维护 Act 1–4 的国际服 / 国服节点名称映射。

当前映射来源：

- `docs/info1.md`
- `docs/info2.md`

- `NodeDetail.tsx`：切换右侧详情 `title`
- `ChapterNav.tsx`：切换左侧流程列表节点名称
